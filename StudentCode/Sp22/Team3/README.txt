1. Please install the relevant libraries in a python3.6+ virtual environment before running the programs:
    - scikit-learn
    - TensorFlow
    - Matplotlib
    - seaborn
    - pandas

2. For data analysis and relevant plots, please run:
    $ python analysis.py

3. For accuracy results from different ML models, please run:
    $ python models.py

4. For accuracy result and training of the neural network, please run:
    $ python ann.py
